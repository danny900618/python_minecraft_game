# 老師專用：解鎖碼對照表

> 上完對應的課程後，把代碼傳給學生（LINE / 訊息均可）。
> 學生在遊戲主選單按 **[U]** 輸入代碼，永久解鎖，不需要換檔案。
> 代碼不分大小寫。

## 傳給學生的訊息範本

**有新內容更新時：**
```
遊戲有更新！
1. 去 GitHub 重新 Download ZIP，解壓縮成新資料夾
2. 把舊資料夾裡的 player_save.json 複製到新資料夾（這樣進度不會不見）
3. 點開始遊戲.bat

新關卡解鎖碼：STONE04
打開遊戲 → 主選單 → [U] 輸入解鎖碼 → 就可以玩新關卡囉！
```

**只有新解鎖碼時（沒改遊戲內容）：**
```
新關卡解鎖碼：STONE04
打開遊戲 → 主選單 → [U] 輸入 → 解鎖！
```

## 難度調整（game_config.json）

如果覺得太難或太簡單，用文字編輯器打開 `game_config.json` 修改：

| 設定 | 預設值 | 說明 |
|------|--------|------|
| `questions_per_chapter` | 5 | 每關出幾題 |
| `max_lives` | 2 | 每次挑戰的生命數 |
| `show_explain_on_wrong` | true | 答錯後顯示解說 |

---

## 重置學生進度

刪除 `player_save.json` 即可完全重置（遊戲重新啟動時會自動建立新的）。
