## if 語法（10 分鐘）

### 最基本的 if
```python
hp = 3
if hp < 5:
    print("血量危險！趕快逃跑！")
```

**重點：**
1. 條件後面一定要加 **冒號 `:`**
2. 下一行要 **縮排（4 個空格）**
3. 縮排的行才屬於 if 的區塊

```
if 條件:
    做這件事（縮排）
    可以有多行（都要縮排）
```

❓ **讓學生試試忘記縮排** → 看 IndentationError

### if ... else
```python
hp = 10
if hp < 5:
    print("危險！逃跑！")
else:
    print("血量還可以，繼續探索！")
```

`else` 不需要條件，當 `if` 不成立時執行。

### if ... elif ... else
```python
hp = 10
if hp >= 15:
    print("滿血！攻擊！")
elif hp >= 5:
    print("血量正常，繼續戰鬥")
elif hp >= 1:
    print("血量危險！逃跑！")
else:
    print("你死了...")
```

> **類比：** elif 就像 Minecraft 的多重條件電路——一個接一個判斷，第一個成立就執行，其他跳過

## 比較運算子（5 分鐘）

| 運算子 | 意思 | 範例 |
|--------|------|------|
| `==` | 等於 | `hp == 20` |
| `!=` | 不等於 | `hp != 0` |
| `>` | 大於 | `level > 10` |
| `<` | 小於 | `hp < 5` |
| `>=` | 大於等於 | `level >= 20` |
| `<=` | 小於等於 | `hp <= 0` |

❗ 常見混淆：`=` 是賦值，`==` 是比較！

```python
hp = 20          # 把 20 放進 hp（賦值）
if hp == 20:     # 比較 hp 是不是 20
    print("滿血！")
```

## 縮排常見錯誤

```python
# ❌ 忘記縮排
if hp < 5:
print("危險")       # IndentationError!

# ❌ 縮排不一致
if hp < 5:
    print("危險")
  print("逃跑")     # IndentationError!（2格 vs 4格）

# ✅ 正確
if hp < 5:
    print("危險")
    print("逃跑")
```
